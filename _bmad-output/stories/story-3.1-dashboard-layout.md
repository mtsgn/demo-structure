# Story: Dashboard Layout & Navigation

## Description

Create the main dashboard layout with sidebar navigation, header with user profile, and a responsive grid system for widgets.

## Acceptance Criteria

- [ ] Sidebar navigation with collapsible menu
- [ ] Header with user avatar and notifications
- [ ] Responsive grid layout (2, 3, or 4 columns based on screen)
- [ ] Dark/light theme toggle
- [ ] Breadcrumb navigation

## Technical Notes

- Use CSS Grid for layout
- Implement with React components
- Store theme preference in localStorage
- Mobile-first responsive design
